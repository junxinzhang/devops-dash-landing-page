FAQ

Common questions about DevOps Dashboard.

Which CI/CD tools are supported?
Native support for AWS CodeBuild and CodeDeploy, can also integrate with Jenkins, GitLab CI, GitHub Actions and other mainstream CI/CD tools via API.
Is big screen display supported?
Yes. Dashboard uses autofit.js specifically designed for big screen display, supports auto-adaptation to different resolutions, provides real-time refreshing visual dashboard.
What is the data refresh rate?
Default auto-refresh every 5 minutes to ensure displayed information is always current. Refresh rate can be configured based on requirements.
How to deploy?
Project is built with Vite, supports AWS CodeBuild + CodeDeploy automated deployment, can also deploy to any platform supporting static sites like Nginx, S3, etc.
